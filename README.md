# 📊 Excel Call Centre Dashboard 2023

An interactive and dynamic Excel dashboard built to analyze call centre 
performance data for the year 2023. This project covers the complete 
data analysis workflow — from raw data cleaning to an executive-level 
dynamic dashboard — using advanced Excel features.

## 🎯 Project Objective

To transform raw call centre data into actionable business insights 
using Excel's advanced features, enabling stakeholders to monitor 
KPIs, agent performance, and customer satisfaction trends 
interactively through a dynamic dashboard.

## 📈 Dashboard Highlights

- **Call Trend** — Monthly call volume chart (Jan–Dec)
- **Day-wise Analysis** — Busiest days identified (Saturday: 161 calls)
- **Monthly Trend** — Pie chart with month-wise call distribution
- **Agent Performance** — Calls and revenue per representative (R01–R05)
- **Gender Analysis** — Male vs Female caller split across cities
- **Customer Ratings** — 1 to 5 star rating distribution
- **KPI Cards** — Live summary tiles for all key metrics
- **Dynamic Slicers** — Filter entire dashboard by Representative and Month

## 💡 Key Insights from the Data

- **Saturday** records the highest call volume (161 calls)
- **R02** handled the most calls (218 total)
- **R03** generated the highest revenue ($20,872)
- **Cleveland** has the largest male caller base (326 vs 63 female)
- **4-star rating** is the most common customer feedback (428 counts)
- **August** appears to be the peak month based on monthly trend

## 🚀 How to Use

1. Download both `.xlsx` files from this repository
2. Open `EXCEL-PROJECT-CALL CENTRE REPORT 2023.xlsx` in 
   Microsoft Excel (2016 or later recommended)
3. Enable editing and data connections if prompted
4. Navigate to the **Dashboard sheet**
5. Use the **slicers** to filter by Representative or Month dynamically
